## C.2 Scenario Universe Builder

### Event Detection Engine

```python
class EventClusteringEngine:
    """Identifies events from stream metadata and viewer activity."""
    
    def detect_events(self, active_streams: List[Stream]) -> List[Event]:
        # Cluster by geolocation
        geo_clusters = self.cluster_by_location(
            streams=active_streams,
            radius_meters=500,
            min_streams=3
        )
        
        # Cluster by content similarity
        content_clusters = self.cluster_by_content(
            streams=active_streams,
            similarity_threshold=0.7,
            min_streams=2
        )
        
        # Merge clusters and classify
        events = self.merge_clusters(geo_clusters, content_clusters)
        
        for event in events:
            event.type = self.classify_event(event)
            event.severity = self.assess_severity(event)
            event.verified = self.verification_score(event)
        
        return events
```

- ✅ Can: Flag content, apply temp restriction (max 24h), send warning
- ❌ Cannot: Permanently ban, delete content, access payout data
